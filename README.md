## 🎨 NeoCloud — это...

**NeoCloud-Desktop** — современный форк SoundCloud Desktop с **фиолетово-черным дизайном**, оптимизированный для ночного прослушивания и максимального стиля.

```
✨ НОВЫЕ ФИЧИ НАШЕГО ФОРКА:
-  Фиолетово-черная тема (dark purple aesthetic)
-  Улучшенная анимация переходов  
-  Минималистичный интерфейс
-  Оптимизация под слабые ПК
```

**Базовые возможности** (от upstream):
- Работает в России без VPN
- Без рекламы и капчи  
- Нативный Tauri 2 (~15 МБ, 80 МБ RAM)
- Медиа-кнопки + Discord Rich Presence

## 🛠 Сборка из исходников

```bash
git clone https://github.com/cursedacad3my/NeoCloud-Desktop.git
cd NeoCloud-Desktop/desktop
pnpm install
pnpm tauri dev
```

**Требования:**
- Node.js 22+
- pnpm 10+ 
- Rust 1.77+
